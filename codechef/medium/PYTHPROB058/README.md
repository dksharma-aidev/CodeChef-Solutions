## Problem

### Worked example - Print multiple items

In this example, we practice using the `print()` function to combine multiple items in one line. The goal is to create a greeting that includes a user’s name and the current year.

For instance:

- User’s name: "Alice"
- Current year: 2023
- Final message: "Hello Alice, welcome to the year 2023!"

When executed, the code will output:

```
Hello Alice, welcome to the year 2023 !

```

Python automatically adds a space between each item, making it easy to create nicely formatted messages. You can experiment by changing “Alice” to a different name or “2023” to another year to see how the output updates!

## Solution

**Language:** Python  
**Runtime:** N/A  
**Memory:** N/A  
**Submitted:** 2026-07-24T05:59:59.225Z  

```py
# Printing multiple items in a single statement, including strings and numbers.

# The print() function combines all arguments and separates them with a default space.

# In this example, a greeting message, name, and year are printed together seamlessly.

print("Hello", "Alice,", "welcome to the year", 2023, "!")

```
